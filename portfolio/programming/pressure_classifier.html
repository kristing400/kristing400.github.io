Pressure Classifier Experimental Capture | Spring 2017

Collaboration with Jason Ma

Initial Ideas

We were inspired by the potential of what the SenselMorph, a pressure sensor module, could do. With the SenselMorph we will be able to identify not only the physical shapes and surfaces of the objects that touch the sensor, but also the distribution of pressure. With this idea in mind, we wanted to investigate how “identifiable” certain aspects of our body can be from others. The project was in part an exploration of machine learning, as well as what kind of new use cases we could project onto it.

Inspiration

I was inspired by Philipp Schmitt’s Camera Restricta, and wanted to use a similar method of crowd sourcing data while using geolocations. Therefore, I made a machine that given a latitude longitude coordinate and a picture taken at the location, will crowd source pictures that people have posted online and fuse them to create a single picture of the location. Thus, this is a camera that takes pictures of what other people has taken.

Collecting Data

First, I wrote a Python script using Temboo/Flikr API to extract all the information I needed given a latitude longitude coordinate as input. I was able to get over thousands of images with one coordinate if the location is a popular spot, which was exactly what I wanted. I then wrote another Python script to download all these images and save it somewhere in my laptop.

Finding Relevant Images: ofxTSNE

This was when I realized that it wasn’t realistic to use all the pictures in one location, because people upload such drastically different pictures. I had to figure out a way to "choose" the pictures that looked closest to the “picture I am taking”. With my professor's help, I was able to use the Openframeworks add-on ofxTSNE to create a grid that sorts all the photos by similarities.

Generating Images & Final Results

Finally I used a 7×7 grid to find the closest 49 pictures in the TSNE grid from a given image (the initial image that I took), and put these batch of pictures into Autopano, a image stitching software. The results turned out better than I thought. Some were fairly normal, and just looked like a shitty panorama job, and others turned out super interesting.
