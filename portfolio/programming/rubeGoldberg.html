Virtual Rube-Goldberg Machine Experimental Game Design | Fall 2016

Collaboration with Jason Ma

For this project, we wanted to create an immersive experience where the user can interact with his/her surroundings. We decided to create a Rube-Goldberg Machine in Unity. Unlike a usual Rube-Goldberg Machine, the user would need to interact with the piece in order to triger some leverages to finish the whole cycle.
